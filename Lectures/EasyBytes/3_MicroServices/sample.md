## Highlighting the word "text" in Markdown

1. Inline code:

   Renders as: `text`

2. Bold:

   ```
   **text**
   ```

   Renders as: **text**

3. Italics:

   ```
   *text*
   ```

   Renders as: _text_

4. Bold + Italics:
   ```
   ***text***
   ```
   Renders as: **_text_**
